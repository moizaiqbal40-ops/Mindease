# MindEase 🧠
AI Mental Health Chatbot using 3 ML models

## What it does
Users describe their stress — app predicts severity using 
Linear Regression, SVM, and Decision Tree, then gives response.

## Tech Stack
Python | Scikit-learn | Streamlit | NLP

## How to run
pip install -r requirements.txt
streamlit run app.py
