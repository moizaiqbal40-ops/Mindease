# 🧠 MindEase

### Understanding stress through language — thoughtfully, transparently, and safely.

An AI-assisted mental wellness companion that estimates stress from natural language using three machine learning models built entirely from scratch with NumPy. Instead of stopping at prediction, MindEase combines engineering with psychology-informed guidance while recognizing the limits of what software should and shouldn't do.

# The Problem

Stress often builds quietly.

Most existing solutions sit at opposite ends of the spectrum:

- simple wellness apps that provide generic advice,
- or professional clinical assessments that require time, appointments, and expertise.

There is very little in between.

What people often need first is a lightweight tool that listens to what they're experiencing, reflects on it responsibly, and encourages healthy next steps without pretending to replace professional care.

---

# The Solution

MindEase allows users to describe how they're feeling in their own words.

Instead of relying on a single prediction model, three independently implemented machine learning algorithms analyze the same input from different perspectives before producing an overall stress estimate.

Every prediction is then paired with psychology-informed guidance based on Cognitive Behavioural Therapy (CBT), mindfulness techniques, and positive psychology principles.

Most importantly, MindEase is designed with a safety-first mindset. If potentially dangerous language is detected, the application deliberately bypasses normal prediction and immediately prioritizes supportive crisis resources instead of attempting to generate a confidence score.

The goal isn't to diagnose mental health.

The goal is to build a transparent engineering system that demonstrates how machine learning can support wellbeing responsibly.
